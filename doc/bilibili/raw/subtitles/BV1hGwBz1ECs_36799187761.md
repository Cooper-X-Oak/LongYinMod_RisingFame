大家好
这里是初夏
今天教大家如何在龙印利记城安装unity explore
背部艾克斯
我试了很多版本
基本上没有可用的
所以我们只能选择melon loader作为模组基础框架
我们在melon loader github releases页面
直接下载最新版的X64zip版本
然后打开游戏根目录解压刚才下载的压缩包
解压完毕后
直接在steam游戏页面启动游戏
游戏启动后
我们可以看见melon loader框架的控制台
我们等待启动的时候
顺便下载一下unity explorer
下载第一个ILICVP版本
Unity explorer
要在我视频中的仓库下载
然后解压压缩包
或者把里面的文件复制出来也一样
我们运气很好
刚好赶上了melon loader
完成了首次出席化
不然我们需要重启一下游戏
来加载unity explore模组
现在我们可以看到unity explore界面显示出来了
并且是可以正常工作的
大家可以通过unity explore来研究一下游戏的功能
以及数据存放在哪些类里面
